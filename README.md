# car-sales-analysis
## Project Overview
This project analyzes car sales data to identify key factors affecting sales performance across manufacturers and models.

## Objective
To answer the main question:
**What affects car sales the most?**

## Tools Used
- Power BI
- Excel / Google Sheets

## Dataset
Car_sales.csv (158 rows, multiple features including price, horsepower, fuel efficiency, and resale value)

## Dashboard
Includes:
- 4 scatter plots (Sales vs key variables)
- Top models and manufacturers analysis
- Price segmentation

## Data Cleaning
- Spelling errors checked
- Missing values handled using the median
- Retention Rate added
- Data types corrected
- Dots changed to commas
  
## Conclusion
The analysis highlights the primary drivers of car sales and provides insights for optimizing product strategy.
